# ConsmicLexicon.Foundation.Globalization

## Overview

ConsmicLexicon.Foundation.Globalization provides culture-aware operations and localization utilities for .NET applications. It focuses on proper handling of cultural differences in data formatting and processing.

## Features

- Culture-aware operations
- Localization utilities
- Format providers
- Resource management
- Date and time handling
- Number formatting

## Key Components

### Culture Operations
- Culture-aware comparisons
- Format providers
- Resource handling

### Localization
- Resource management
- String localization
- Message formatting

### Formatting
- Number formatting
- Date/time formatting
- Custom format providers
